# HiddenGem Frontend

React-based frontend for the HiddenGem quantitative trading system.

## Setup

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
```bash
cp .env.example .env
# Edit .env with your configuration
```

## Development

### Run development server:
```bash
npm run dev
```

The application will be available at http://localhost:5188

### Build for production:
```bash
npm run build
```

### Preview production build:
```bash
npm run preview
```

### Type checking:
```bash
npm run type-check
```

### Linting:
```bash
npm run lint
```

### Code formatting:
```bash
npm run format
```
